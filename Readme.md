## Asteroids
The 1979 Atari classic, remade in Love2D using Lua.
  
### Requirements:
#### Arch Linux
```
pacman -S love
```
#### Mint / Ubuntu
```
apt-get install love
```
#### Windows
[Love2D](https://love2d.org/)

### Running the game:
#### Linux
```
git clone https://github.com/Ithalos/Asteroids.git Asteroids
cd Asteroids
love .
```
